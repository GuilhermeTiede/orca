## Sistema Web para Agência de Viagens

<br>

*Below we will have the description in English and Portuguese*

<br>

<h3>Este projeto visa desenvolver um sistema web completo para uma agência de viagens, oferecendo funcionalidades de gerenciamento de reservas, pacotes turísticos, clientes e muito mais. A interface do usuário, já desenvolvida em React, proporciona uma experiência intuitiva e responsiva. O back-end robusto, construído com PHP/Laravel, garante a segurança e eficiência das operações.</h3>

<br>

<h2>Requisitos da Vaga</h2>

<br>

<h3>**Técnicos:**</h3>

<ul>
<li>PHP: Sólida compreensão da linguagem e suas nuances.</li>
<li>Laravel: Experiência com o framework, incluindo roteamento, controladores e modelos.</li>
<li>JavaScript e React: Domínio da biblioteca para criação de interfaces dinâmicas.</li>
<li>MySQL: Conhecimento em modelagem de dados e execução de consultas.</li>
<li>InertiaJS: Familiaridade com o adaptador para otimizar a comunicação entre front-end e back-end.</li>
</ul>

<br>

<h3>**Experiência:**</h3>

<ul>
<li>Desenvolvimento web comprovado, com foco em projetos de médio a grande porte.</li>
<li>Experiência com integração de APIs e requisições web.</li>
<li>Conhecimento em Git para versionamento de código.</li>
</ul>

<br>

<h3>**Diferenciais:**</h3>

<ul>
<li>Experiência com testes automatizados (PHPUnit, Jest).</li>
<li>Conhecimento em Docker para criação de ambientes de desenvolvimento consistentes.</li>
<li>Inglês técnico para leitura de documentação e comunicação em fóruns.</li>
</ul>

<br>

<h2>Sobre o Projeto</h2>

<h3>O projeto encontra-se em andamento, com o front-end (React) e parte do back-end (PHP/Laravel) já desenvolvidos. A estrutura do projeto segue as melhores práticas, com código limpo, organizado e documentado em inglês. As funcionalidades de CRUD (criar, ler, atualizar e deletar) já estão implementadas, assim como algumas regras de negócio específicas do domínio da agência de viagens.</h3>

<br>

<h3>**Estrutura do Projeto:**</h3>

<ul>
<li>Front-end: React, componentizado e estilizado, pronto para ser integrado ao back-end.</li>
<li>Back-end: PHP/Laravel, com models, controllers e migrations definidas.</li>
<li>Banco de Dados: MySQL, com esquema otimizado para o sistema.</li>
</ul>

<br>

<h3>**Próximos Passos:**</h3>

<ul>
<li>Integrar o front-end (React) com o back-end (Laravel) utilizando InertiaJS.</li>
<li>Implementar as demais regras de negócio e funcionalidades do sistema.</li>
<li>Realizar testes completos para garantir a qualidade e estabilidade do sistema.</li>
<li>Otimizar o desempenho e a segurança do sistema.</li>
</ul>

## Web System for Travel Agency

<br>

<h2>Job Requirements</h2>

<br>

<h3>**Technical:**</h3>

<ul>
<li>PHP: Solid understanding of the language and its nuances.</li>
<li>Laravel: Experience with the framework, including routing, controllers, and models.</li>
<li>JavaScript and React: Proficiency in the library for creating dynamic interfaces.</li>
<li>MySQL: Knowledge of data modeling and query execution.</li>
<li>InertiaJS: Familiarity with the adapter to optimize communication between front-end and back-end.</li>
</ul>

<br>

<h3>**Experience:**</h3>

<ul>
<li>Proven web development experience, focusing on medium to large projects.</li>
<li>Experience with API integration and web requests.</li>
<li>Knowledge of Git for code versioning.</li>
</ul>

<br>

<h3>**Differentials:**</h3>

<ul>

<li>Technical English for reading documentation and communicating in forums.</li>
